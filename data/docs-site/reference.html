Tournaments — RPC Reference

Source of truth: data/modules/src/tournaments/rpcs.ts. All 30 RPCs are registered on every Nakama pod. Method is POST unless otherwise noted; payload is always a JSON-encoded string.

Common request bodies

tournament_enter

// Request
{
  "slug": "gk-royale-daily",
  "idempotency_key": "uuid-v4",       // required, unique per intent
  "ref_code": "AB12CD"                 // optional referral attribution
}
// Response (success)
{
  "ok": true,
  "entry_id": "ent_...",
  "pot_bc": 5050,
  "your_rank": 1,
  "founder": false,
  "idempotent": false                  // true if dedup'd
}
// Response (errors — non-exhaustive)
{ "ok": false, "error": "tournament_not_open", "status": "PRE_ENROLL" }
{ "ok": false, "error": "insufficient_balance", "have": 30, "need": 50 }
{ "ok": false, "error": "entry_blocked_in_us_state", "state": "NY" }
{ "ok": false, "error": "country_not_allowed", "country": "FR" }
{ "ok": false, "error": "age_gated" }
{ "ok": false, "error": "rate_limited", "retry_after_sec": 12 }

Realtime stream — tournament:{slug}

Subscribe via socket.joinChat("tournament:gk-royale-daily", 1, false, false). Messages are JSON in msg.content:

// op codes
1001  entry         { op:1001, slug, user_id, pot_bc, entries_count }
1002  submit        { op:1002, slug, user_id, score, your_rank }
1003  elimination   { op:1003, slug, eliminated_user_ids:[...], round, survivors }
1004  settled       { op:1004, slug, payouts:[{user_id,rank,bc}], rake_bc }
1005  cert_ready    { op:1005, slug, user_id, url }

Error codes — common patterns

ErrorCauseClient should
tournament_not_openSlug status is not OPEN/ACTIVEShow "Opens at <ISO>" CTA
tournament_closedSlug is SETTLING/SETTLEDSend to results screen
insufficient_balanceWallet < entry_fee_bcOpen Buy-BC sheet
entry_blocked_in_us_stateState is in ENTRY_BLOCK_US_STATESShow AMOE upsell
redemption_blocked_in_us_stateState is in REDEMPTION_BLOCK_US_STATESAllow play, block redeem
country_not_allowedCountry not in countries_allowedShow country-specific copy
age_gatedDOB < 18 years agoHard block
rate_limitedBurst over limitBackoff retry_after_sec
idempotency_conflictSame key used with different payloadGenerate a fresh UUID
pick_window_closedSubmitting picks after lockShow "Picks locked" UI
learning_incompleteAMOE without 6/6 videosSend to learning track
founder_cap_reachedPre-enroll past 1000 capShow "you're enrolled (no founder bonus)"
bracket_unreachableService degradedRetry in 30s
