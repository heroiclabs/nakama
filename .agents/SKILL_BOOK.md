## Hard Rules

1. **NEVER hand-edit `index.js`** — it is 100% generated by postbuild.js
2. **NEVER use Node built-ins** — no `require('fs')`, `require('http')`, etc.
3. **NEVER use global variables for state** — Goja VM pool resets context
4. **TypeScript target MUST be es5** — Goja does not support ES6+
5. **All RPC functions MUST be in global scope** — no lambda assignments
6. **All env vars for JS must be in RUNTIME_ENV_KEYS** in docker-compose.yml
7. **Rebuild + restart after any code change** — modules are loaded at startup
